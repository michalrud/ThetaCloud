ThetaCloudHumidity
==================

.. cpp:class:: ThetaCloudHumidity

	Humidity and temperature sensor integrated in main ThetaCloud board.

	:Publishes:

		:humidity:
			Measured humidity.

			:units:	``[%RH]``
			:format: ``float``
		:temperature:
			Measured temperature.

			:units:	``[C]``
			:format: ``float``

	:Consumes: This module does not consume any data.
